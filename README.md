Название проекта: SkyproWallet

Учебный проект по управлению личными финансами. Реализован на React.

Инструкция по запуску

1. Установите зависимости: `npm install`
2. Запустите проект в режиме разработки: `npm run dev`

Результаты

- Реализация адаптивной верстки Login и Register по макетам.
- Адаптивные страницы Login, Register и Главная страница (MainPage) в полном соответствии с макетом Figma.
- Управление расходами:
  Получение списка транзакций с сервера.
  Добавление новых расходов с выбором категории.
  Удаление существующих транзакций.
